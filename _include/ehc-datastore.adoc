.AWS/VMC
--
AWS 支援以下配置的NetApp儲存：

* FSx ONTAP作為來賓連接存儲
* Cloud Volumes ONTAP (CVO) 作為來賓連接存儲
* FSx ONTAP作為補充 NFS 資料存儲
